# we-movies

## Requirements
- PHP 7.4
- SF 5.3 (latest stable)
- Bootstrap 5
- A free API key from [The Movie Database](https://www.themoviedb.org/)

## Install
```bash
composer install
```

```bash
yarn install
```

```bash
yarn build
```

- Fill "THEMOVIEDB_APIKEY" in .env.dev with your personal API key from [The Movie Database](https://www.themoviedb.org/)
